3D Point Plotter

A flexible 3D point plotter utility that allows for the easy plotting and viewing of up to thousands of points. The plotter features a changable axis and point size to accommodate a dataset of any scale.

This project uses:
  • three.js
  • JavaScript
  • Bootstrap
  • HTML

Bitcoin Mining Calculator

An easy to use utility used to calculate a Bitcoin miner's potential profits in BTC and multiple fiat currencies. The calculator fetches price and Bitcoin network data from the internet and only requires the hash rate (speed of mining) from the user. A projected future profit chart is created dynamically and displayed instantly.

This project uses:
  • AngularJS
  • jQuery
  • Chart.js
  • JavaScript
  • Bootstrap
  • HTML
