My Crypto Buddy

A powerful web application that enables cryptocurrency miners to maximize their profits by calculating and displaying the most profitable coins to mine for their given hardware setup as well as future profitability projections. The app is very light-weight, built using the MEAN stack and is hosted in its entirety on a single-core 2 GB micro-server, running flawlessly, despite seeing over 7,000 daily page views.

This project uses:
  • MongoDB
  • Express Web Server
  • Node.js
  • AngularJS

Universal Currency Converter

The Universal Currency Converter is a simple yet powerful utility that allows for the conversion of any currency to any other currency. This includes over 40 fiat currencies and 225 cryptocurrencies. The tool uses powerful JavaScript in the frontend and backend as well as socket connections to provide instant and live-updating currency conversions without the need to ever refresh the page.

This project uses:
  • A ton of data scraping
  • MEAN Stack
  • Socket.io

Pure JavaScript 3D Engine

A very simple engine that builds 3D objects and displays them from a variable vantage point. Built entirely in JavaScript with no additional libraries. The purpose of this project was to better understand the mechanics of more advanced graphics libraries.
